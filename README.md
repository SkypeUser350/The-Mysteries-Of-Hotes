# 📁 The Mysteries of Hotes

An atmospheric, retro-hacker defense game built using **HTML5, CSS3, JavaScript (ES6), and JSON**. Playable directly from your browser via GitHub Pages.

> **SYSTEM ALERT:** You have booted up an experimental operating system built by a legendary, missing developer named **Hotes**. Crucial source data across Sector 01 to 100 has mutated into hostile anomalies. To solve his disappearance and salvage the mainframe, you must execute diagnostic wipes and purge the incoming corruption.

## 👁️ Gameplay Mechanics

This project blends the level-progression suspense of **DOORS**, the reactive mouse combat of **Combat Initiation**, and the lane-based fortification strategies of **Plants vs. Zombies**.

*   **Grid Quarantine (PvZ Layer):** Glitch files walk left across 3 system network lanes. Deploy specialized structural codeblocks like the `HotesShield` to dynamically halt their approach.
*   **Direct Click Purge (Combat Layer):** Your cursor acts as a hardwired diagnostic sub-routine. Manually click on glitches to chip away at their structural integrity points.
*   **Sequential Sectors (DOORS Layer):** Clear all systemic threats inside the active buffer array to safely flash memory and boot into the subsequent sector. Reach **Sector 100** to reveal the terminal data logs.
